Pygame Puzzle-Adventure Game

**Python ‚Ä¢ Pygame ‚Ä¢ Object-Oriented Programming (OOP) ‚Ä¢ Algorithms**

This project is a **2D grid-based puzzle game** developed in **Python** using the **Pygame** library. It is designed to demonstrate mastery of **Object-Oriented Programming (OOP)** concepts, modular level design, and the implementation of complex, interconnected game mechanics.

The player progresses through a series of levels that sequentially introduce new objects and rules, culminating in a final stage that integrates all learned mechanisms.

‚öôÔ∏è Key Technical Highlights

Feature Technical Implementation
**Modular Architecture** Use of clean OOP classes (Level, Player, Mirror, Gear, etc.) with a base Level class to ensure **separation of concerns** and ease of extension.
**Reflection Logic** Implementation of a dynamic **Light Beam** system (LightBeam) that calculates its path in real-time, using the mirror's **angle** attribute (45¬∞ or 135¬∞) to determine the reflection direction.
**AI and Pathfinding** The **Chaser Enemy** (ChaserEnemy) utilizes a search algorithm (A-star or BFS type) to track the player, dynamically accounting for all obstacles (walls, rocks, mirrors) in its path calculations.
**Complex Interactions** Handling collisions for inter-object interaction: pushing **Boulders** onto **Pressure Plates**, which in turn trigger the activation of other mechanisms (like the light beam).
**Dynamic Hazards** Implementation of timed hazards (like **Bridges** that alternate between solid and open) and collision hazards (rotating **Gears**), requiring precise management of game timing (cooldowns and timers).

üåü Level Overview

The game features **8 progressive levels** (from Level 3 to Level 8), where the final stages test the player's ability to combine all the established rules.

Level Primary Mechanism Introduced
**Level 4** **Mirrors** and **Light Beam** (Spatial logic).
**Level 5** **Chaser Enemy** (Evasion and pathfinding).
**Level 6** **Boulders** and **Pressure Plates** (Object manipulation and logic gates).
**Level 7** **Gears** (Timing hazard and movement precision).
**Level 8** **Synthesis** (Integration of all mechanics into a complex final puzzle).
